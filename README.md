# ChildSafetyApp "Your child, always within reach 💛"
Guardian is a real-time child safety and monitoring mobile application built using Flutter and Firebase. The app enables parents to track their child's live location, visualize risk-prone areas (red zones), and receive instant alerts in case of emergencies. 

# Guardian – Child Safety & Monitoring App

Guardian is a full-stack mobile application developed using Flutter and Firebase, designed to enhance child safety through real-time monitoring and intelligent alert systems.

## 🚀 Features

- 📍 Live GPS Tracking  
  Track the real-time location of children on an interactive map.

- 🔴 Red Zone Detection  
  Visualize high-risk areas using geofenced zones stored in Firestore.

- 🚨 SOS Alerts  
  Instantly notify guardians in emergency situations.

- 🔔 Smart Alerts System  
  Receive alerts when a child enters unsafe areas or triggers SOS.

- 👨‍👩‍👧 Child Profile Management  
  Add, edit, and manage multiple child profiles with ease.

- 🧭 Interactive Map Interface  
  Built using Flutter Map for smooth and responsive navigation.

- ☁️ Firebase Integration  
  Uses Firebase Authentication and Cloud Firestore for real-time data syncing.

- 🧪 Demo + Real Data Support  
  Displays demo alerts when no real data is available, improving UX.

## 🛠 Tech Stack

- Flutter (Frontend)
- Firebase Authentication
- Cloud Firestore (Database)
- Geolocator (Location services)
- Flutter Map (Map rendering)

## 💡 Key Highlights

- Real-time data streaming using Firestore
- Dynamic UI updates based on backend changes
- Clean and modern UI/UX design
- Scalable architecture for future enhancements

## 🎯 Purpose

This project was built to explore real-time systems, mobile app development, and safety-focused applications. It demonstrates how technology can be used to improve personal safety and awareness.
